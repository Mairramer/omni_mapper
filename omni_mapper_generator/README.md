# Omni Mapper Generator

Code generator for the `omni_mapper` package. This package automatically generates type-safe data mappers, eliminating boilerplate code for mapping between Data Transfer Objects (DTOs), Entities, and Models.

## Installation

Add this package as a `dev_dependency` alongside `build_runner`, and add `omni_mapper` to your normal dependencies.

```yaml
dependencies:
  omni_mapper: ^0.1.0

dev_dependencies:
  build_runner: ^2.4.0
  omni_mapper_generator: ^0.1.0
```

## Usage

For detailed usage instructions, please refer to the [omni_mapper package documentation](https://pub.dev/packages/omni_mapper).
